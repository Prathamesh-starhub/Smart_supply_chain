# Smart Supply Chain AI

A React + Vite starter project for building the Smart Supply Chain AI frontend.

## Available Scripts

- `npm install` - install dependencies
- `npm run dev` - start the Vite development server
- `npm run build` - build the production bundle
- `npm run preview` - preview the production build locally

## Project Structure

- `src/` - React source code
- `src/assets/images/` - image assets
- `public/` - public static files
- `index.html` - app entry template

## Notes

This project uses React 18 with Vite for fast development and build performance.
